# `/frontend-quality` – Verificación rápida de calidad del frontend

## Objetivo

Ejecutar los comandos de verificación de calidad más importantes del frontend: lint y typecheck.

## Pasos

1. Ir al directorio del frontend:

   ```bash
   cd /Users/mariorgzr/kaba/kaba-frontend
   ```

2. Ejecutar el linter:

   ```bash
   npm run lint
   ```

3. Ejecutar el typecheck de TypeScript:

   ```bash
   npm run typecheck
   ```
