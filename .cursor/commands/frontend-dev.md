# `/frontend-dev` – Iniciar solo el frontend en modo desarrollo

## Objetivo

Levantar el frontend (Next.js) en modo desarrollo, apuntando al backend ya levantado en `http://localhost:3000`.

## Pasos

1. Asegurarte de que el backend ya esté corriendo (por ejemplo con `/backend-dev`).

2. Ir al directorio del frontend:

   ```bash
   cd /Users/mariorgzr/kaba/kaba-frontend
   ```

3. Iniciar el frontend en modo desarrollo:

   ```bash
   npm run dev
   ```

4. Abrir el navegador en:

   - `http://localhost:9002`
